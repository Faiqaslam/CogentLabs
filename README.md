# CogentLabs
Task for Generative AI Engineer

## Getting Started

### Prerequisites
python3.10
jupyter notebook

### Installation
1. Clone the repo

```sh
git clone https://github.com/Faiqaslam/CogentLabs.git
```
2. Navigate to the project directory
```sh
cd CogentLabs
```
3. Create a virtual enviroment for the project
```sh
## For Windows

python -m venv myenv
```
```sh
## For macOS/Linux

python3 -m venv myenv
```
4. Activate the virtual enviroment
```sh
# For Windows
venv\Scripts\activate
# For macOS\Linux
source myenv/bin/activate
```
5. Install required dependencies:
```sh
pip install -r requirements.txt
```
